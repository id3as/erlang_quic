    quic_hkdf (quic v0.6.5)

    View Source

    HKDF implementation for QUIC key derivation.

    HKDF is used in QUIC/TLS 1.3 for deriving keys from shared secrets. This module implements the Extract-Expand paradigm with SHA-256.
